# PayloadGen 🔥

A modular payload generator for web exploitation including XSS, SQL Injection, and Command Injection. Supports encoding and obfuscation.

---

## 🚀 Features

- XSS, SQLi, Command Injection Payloads
- Encoding (URL, Base64, Hex, Unicode)
- Modular CLI Interface
- Output to terminal or file
